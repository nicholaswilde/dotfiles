# Specification - Create Agent Skills Catalog

## Overview
The goal of this track is to document the custom scripts and agent workflows present in this repository, such as the `/app add <path>`, `/app check-configs` commands, and `/app sync <app_name>`.

## Requirements
1. **Catalog file:** Create a new markdown file `.agents/skills_catalog.md` detailing the commands, their usage syntax, dependencies, and behavior.
2. **Conductor Integration:** Link the catalog in the `conductor/index.md` context index to make it easily discoverable.
3. **Verify:** Confirm that markdown rendering and internal links function correctly.
